L'image avec Python

Chargement d'une image avec PIL

La bibliothèque PIL (Python Imaging Librairy) permet de traiter des images avec Python. Elle est conçue de manière à offrir un accès rapide aux données contenues dans une image. Cette bibliothèque est incluse dans l'IDE Edupython.

Pour charger et afficher une image avec PIL :


from PIL import Image,ImageDraw                
img=Image.open("iguane.jpg")
img.show()

L'image nommée iguane.jpg doit être dans le même dossier que le script python. L'instruction de la ligne 3 permet de charger l'image est de l'affecter dans une variable de type Image nommée img.

Définition d'une image

La méthode width appliquée à l'image nommée img retourne la largeur de l'image en pixels et la méthode height retourne sa hauteur en pixel également :


from PIL import Image,ImageDraw                       
img=Image.open("iguane.jpg")
largeur=img.width
longueur=img.height
print("La largeur de l'image est :",largeur)
print("La longueur de l'image est :",longueur)

Charger une image de votre choix et rajouter au programme une instruction qui pemet d'afficher la définition de l'image.

Obtention des composantes RVB d'une image

Repérage d'un pixel

Dans un premier temps, il est nécessaire de connaître les coordonnées du pixel dans l'image. Afin de repérer un pixel dans une image, Python utilise un système d'axe dont l'origine est le coin supérieur gauche de l'image. L'axe des abscisses est orienté vers la droite mais attention, l'axe des ordonnées est orienté vers le bas.

Obtention des composantes RVB du pixel

Afin de déterminer les composantes RVB, on utilise la méthode getpixel() :


couleur=img.getpixel((x,y))
print(couleur)

On peut créer 3 nouvelles variables R, V et B et leur affecter chacune une des 3 valeurs du tuple :


R,V,B=couleur
print("La composante rouge= : ",R)
print("La composante verte :",V)
print("La composante bleue : ",B)

Séparation des trois couches RVB

Afin de séparer les 3 couches RVB de l'image iguane.jpg, nous allons dans un premier temps créer 3 images vierges de type RGB de même largeur et hauteur que l'image d'origine image0. Nous les nommerons imageR, imageV et imageB.


from PIL import Image
image0=Image.open("iguane.jpg)
imageR=Image.new("RGB",(image0.width,image0.height))

Ensuite il faut balayer chaque pixel de l'image d'origine puis :

  • Extraire ses 3 composantes R,V et B (vu précédemment) ;
    • 
 R,V,B=image0.getPixel((x,y))
  • Attribuer la couleur (R,0,0) au pixel correspondant de imageR ;
    • 
imageR.putpixel((x,y)),(R,0,0))
  • Attribuer la couleur (0,V,0) au pixel correspondant de imageV ;
  • Attribuer la couleur (0,0,B) au pixel correspondant de imageB.

Il ne reste ensuite plus qu'à afficher ces 3 images éventuellement à les sauvegarder


imageR.save("imageR.jpg")

Transformation d'une image en niveaux de gris

Une image en niveau de gris est une image dont les 3 composantes R,V,B sont identiques. Afin d'effectuer cette transformation, il faut dans un premier temps créer une image vierge de type RGB de même largeur et hauteur que l'image l'origine imageO. Nous la nommerons imageG. Ensuite il faut balayer chaque pixel de l'image d'origine puis :
  • Extraire ses 3 composantes R,V et B ;
  • Effectuer la moyenne de ces 3 composantes et affecter la valeur moyenne à une variable G.
  • Attribuer la couleur (G,G,G) au pixel correspondant de l'image.
